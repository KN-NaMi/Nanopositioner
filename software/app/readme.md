# NaMi Protocol DAC Controller (TUI)

## Installation and Quick Start

### 1. Create a Virtual Environment

```sh
python -m venv .venv
```

### 2. Activate Environment

First, activate the new environment.

* *On macOS/Linux:*
```sh
source .venv/bin/activate
```

* *On Windows:*
```bat
.venv\Scripts\activate
```

Then, while the environment is active, install the required packages:
```sh
pip install -r requirements.txt
```

### 3. Run the Application

With the environment still active, run the main script:
```sh
python main.py
```
